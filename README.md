# Industry-Immersion-education-chatbot
# 🎓 education-chatbot — Education LLM Chatbot

A **domain-specific AI chatbot** for the **Education & Training industry**, built using modern LLM techniques like **QLoRA fine-tuning, RAG (FAISS), LangChain, and Groq LLaMA-3.3-70B**.

---

## 🚀 Overview

EduBot Pro is an **end-to-end AI project** that demonstrates:

- Fine-tuning a **TinyLlama-1.1B model** using QLoRA  
- Building a **RAG pipeline** with FAISS + LangChain  
- Using **Groq API** for ultra-fast LLM inference  
- Deploying a **chatbot UI with Gradio**  

💡 Runs completely on **Google Colab (Free GPU)**

---

## ✨ Key Features

- 🧠 Fine-tuned LLM (QLoRA — 4-bit efficient training)  
- 📚 RAG-based answers with knowledge grounding  
- 💬 Multi-turn conversation with memory  
- 🔄 History-aware question understanding  
- ⚡ Fast inference using Groq LLaMA-3.3-70B  
- 🌐 Public chatbot link (via Gradio)  

---

## 🛠 Tech Stack

**Python, TinyLlama, QLoRA, FAISS, LangChain, Groq API, Gradio, HuggingFace, Google Colab**

---

## 🔎 How It Works

1. User asks a question  
2. System retrieves relevant context using **FAISS (RAG)**  
3. Query is enhanced using **chat history**  
4. Final answer generated using **Groq LLaMA-3.3-70B**  
5. Response shown in **Gradio chatbot UI**  

---

## 📊 Dataset

- HuggingFace dataset (**52K samples**)  
- Filtered to **education-specific data**  
- + Custom expert Q&A and reference documents  

---

## 🚀 How to Run

1. Open notebook in **Google Colab**  
2. Enable **T4 GPU**  
3. Add your **Groq API key**  
4. Click **Run All**  
5. Open the generated **Gradio link**
